# Archived planning docs

These are historical planning / status / post-mortem documents that were
moved out of the repository root (#5064). They reflect the state of the
project at the time they were written and are **not** kept up to date — they
are preserved here for context only.

For current, living documentation see:

- [`CLAUDE.md`](../../CLAUDE.md) — architecture orientation and build commands
- [`CHANGELOG.md`](../../CHANGELOG.md) — per-version history
- [`CONTRIBUTING.md`](../../CONTRIBUTING.md) — contributor workflow
- [`README.md`](../../README.md) — project overview

Do not treat anything in this directory as a description of the current
codebase. In particular, `BUG-STRING-COMPARISON.md` describes a bug that has
since been fixed (regression coverage lives in
`test-files/test_edge_regression.ts` and `test_string_pointer_semantics.ts`),
and the version/feature claims in `GAPS.md`, `PERF_ROADMAP.md`, etc. are
stale.
